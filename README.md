# Smart Recipe Preparation Agent

## Overview

Smart Recipe Preparation Agent is an AI-powered cooking assistant built using IBM Watsonx.ai and Granite 4h Small.

Users can enter available ingredients and receive:

* Recipe Suggestions
* Step-by-Step Cooking Instructions
* Ingredient Substitutions
* Cooking Tips
* Dietary Recommendations

## Technologies Used

* IBM Watsonx.ai
* Granite 4h Small
* LangFlow
* Streamlit
* Python

## Example Input

egg, onion, tomato

## Example Output

* Recipe Name
* Ingredients Required
* Cooking Instructions
* Ingredient Substitutions
* Cooking Tips
* Dietary Recommendations

## Workflow

User Input → LangFlow → IBM Granite → Recipe Response
